AxS Calculator

Welcome to the AxS calculator. With this application you can calculate the scores with the AxS score format.

Getting started

To get started, you first have to get your API key from osu! and save it in the application. This can be done here, or by clicking on cogwheels () on the left.
After you saved your API key, you are pretty much ready to go. Go to "Create a lobby" in the left sidebar to create a lobby and "Lobby overview" to view all the created lobbies
